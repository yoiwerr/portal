MakeItSpecific — AI 工作流增强 Agent                                         2026.05 – 至今
独立设计并开发的对话式工作流 Agent，基于 LangGraph ReAct Agentic Loop，将模糊需求通过意图识别、追问补全、RAG 检索与工具调用转化为结构化方案，支持 Token 级 SSE 流式输出。

• 设计 8 节点 LangGraph StateGraph 决策流，Planner 基于 LLM JSON mode 评估信息完整度（<75% 自动追问），Checkpoint 语义中枢在每次执行后检查方向偏移并触发修正，与 Reflector 输出质量审查形成双层保障，最多 3 次自动重试。基于 astream_events 实现真 Token 级 SSE 推送，含 tool_start/tool_end 实时事件。

• 设计三层上下文引擎：L1 最近 3 轮原文零延迟保留；L2 LLM 滚动摘要增量更新覆盖全部历史；L3 LLM 结构化提取偏好/决策/约束存入 PGVector，跨会话语义召回。含 keyword 重叠率主题切换检测，自动重置上下文避免旧话题污染。配套 L2 跨会话记忆与 L3 用户画像双层记忆系统。

• 构建 Dense + BM25 混合检索管道：PGVector IVFFlat + PostgreSQL tsvector GIN 双路召回 → RRF 融合 → qwen3-rerank 精排。自研 SemanticChunker，基于相邻句子 embedding 相似度断崖检测自动切分文档，替代固定大小分块。

• 实现多 Provider LLM 工厂（装饰器注册模式，DashScope Qwen-Plus / DeepSeek / OpenAI / Local，Auto 自动选型）。LLM + 规则双通道意图分类器覆盖 6 类场景自动路由。全链路 JSON 解析含多层正则降级，LLM 异常时流程不中断。PostgreSQL 统一承载向量检索、全文检索、会话持久化与记忆存储。
