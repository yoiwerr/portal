MakeItSpecific — AI 工作流增强 Agent                                         2026.05 – 至今
项目描述：独立设计并开发基于 LangGraph ReAct Agentic Loop 的对话式工作流 Agent。将模糊自然语言需求通过意图识别、多轮追问、RAG 知识检索与工具调用，自动转化为结构化可执行方案，支持 Token 级 SSE 流式输出。

•  设计 8 节点 LangGraph StateGraph 决策流，包含 3 条条件路由、LLM JSON mode 信息完整度评分（<75% 触发追问）、Planner 语义中枢（每次执行后检查方向偏移并自动修正）与 Reflector 质量审查双层保障，最多 3 次自动重试。

•  构建 Dense + BM25 混合检索管道：PGVector IVFFlat + PostgreSQL tsvector GIN 双路召回 → RRF 融合 → qwen3-rerank 精排。自研 SemanticChunker 基于相邻句子 embedding 相似度断崖自动切分。设计三层上下文引擎（L1 滑动窗口 + L2 滚动摘要 + L3 语义事实 PGVector 跨会话召回），含主题切换检测自动重置。

•  实现多 Provider LLM 工厂（DashScope Qwen-Plus / DeepSeek / OpenAI / Local，装饰器注册 + Auto 自动选型），LLM 异常时多层正则降级解析保活。设计双通道意图分类器（LLM + 规则），6 类场景自动路由到对应 Skill。

•  设计 4 工具 Agent 工具集，每 Skill 按需分配工具子集；L2 跨会话记忆（摘要→embedding→向量召回）+ L3 用户画像（偏好增量学习）双层记忆系统。PostgreSQL 统一承载向量检索、全文检索、会话持久化与记忆存储。Docker Compose + nginx 反向代理部署。
