MakeItSpecific — AI 工作流增强 Agent                                         2026.05 – 至今
独立设计并开发的对话式工作流 Agent，将用户模糊需求通过自动意图识别、追问补全、知识检索与工具调用转化为结构化可执行方案。

整体架构：基于 LangGraph 构建多节点有状态决策流，实现需求完整性自动评估（不足时触发追问）、多轮对话状态持久化与自动回退重试，确保从模糊输入到结构化输出的稳定收敛。

三层记忆系统：设计 L1 原始窗口 + L2 滚动摘要 + L3 语义事实三层互补的上下文管理机制，解决长对话中信息遗忘与上下文漂移。L3 支持跨会话提取用户偏好与历史决策并在新对话中自动召回。含主题切换检测，话题变化时自动重置记忆。

RAG 检索管道：构建 Dense + BM25 双路混合检索链路，通过 RRF 融合与 Rerank 精排实现领域知识的精准召回与上下文注入。实现语义分块器，基于相邻句子 embedding 相似度自动识别文档断点，保留段落语义完整性。

意图识别与工作记忆：LLM + 规则双通道意图分类，覆盖 6 类场景自动路由。已确认意图与需求维度锁定为工作记忆跨轮注入，防止多轮对话中方向漂移与信息丢失。

审查监管：执行后自动触发语义方向与输出质量双重校验，检测偏离时自动修正并重试。全链路 LLM 输出含多层降级解析，模型异常时流程不中断。

工具边界裁定：基于 LLM 与工具的实际能力边界，为每类场景设定差异化工具子集与调用权限，避免模型在不匹配场景下越权使用或选择困难。制定工具与上下文工程边界规范文档，约束 Agent 行为边界与安全策略。
